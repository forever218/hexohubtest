赛博朋克2077是我玩的最久、最喜欢的3A大作游戏,里面的艺术设计和世界观都让我非常欣赏。玩了很久之后，我对于事物的一些看法也有了不小的改变。2077于我而言的意义早已超过游戏本身。想来想去，是时候专门为2077写一篇文章了。
{% asset_img 1.jpg cyberpunk_2077 %}
{% note info flat %}
在我看来，2077与其说是一个游戏，更不如说是一个宏大的世界。所以仅仅以“游戏内容”来定义接下来的文字，可能会显得很勉强。2077的世界太大了，大的令人难以想象，在短短的篇幅里不可能装得下所有内容。
{% endnote %}
具体的游戏剧情可以看这篇文章：{% btn 'https://www.zhihu.com/question/434118625/answer/1630001326?utm_source=zhihu&utm_medium=social&utm_oi=1009341149630246912',地址,far fa-hand-point-right,red larger %}.简单概括一下，就是一个平凡人在赛博朋克这样一个`高技术、低生活`的环境里的挣扎与奋斗。游戏的主角是V，一开始是个很不起眼的街头混混，一次机缘巧合，和荒坂公司扯上了关系（中间人的任务），当时谁都以为，这不过是一次高风险高回报的偷盗任务罢了，可是谁也没想到，当V接过荒坂塔的信息芯片时，一切都变得愈发不可收拾......
游戏玩法上面没有什么可圈可点的地方，不过看得出来，制作组想创造一个`智力与武力`并存的战斗系统。如果你是黑客技术爱好者，你可以不用开枪、不用流血的完成每一个任务；如果你是刀剑迷，你也可以体验拿着武士刀从头砍到尾的快感；如果你是FPS玩家（这应该是游戏本来的目的...），第一人称射击做的也很不错。
{% asset_img 2.jpg 通过黑客手段击杀敌人 %}
{% asset_img 3.jpg 第一人称射击 %}
剧情方面，我觉得比起最初制作组画的大饼和夜之城的这个设计，主线剧情就显得过于简陋了。因为感觉这么大的城市，能和主线关联上的内容并不多，主角一路在与内心作斗争的同时，不断克服各种各样的困难，最终干死了亚当重锤，然后主线就结束了。再后面就是两条小支线，要么选择自杀，要么选择上空间站去治疗（最后还能选择是否变成数字生命，还是回到地球度过剩下的3个月）。请允许我用近乎于倒叙的方法，从剧情的结尾开始说起......
{% asset_img 4.jpg cyberpunk2077城市风景 %}
主线中，最让我感动的是V从月球回到地球后的那段剧情：左边在放制作人员名单，右边是轮流播放的全息电话。之前和V关系好的朋友、中间人、义体医生......依次出现，给V送上最后的话语，因为他们都知道，3个月后，V就会离开这个世界了。有祝福的，有沮丧的，但是都特别符合游戏中各个人物的设定。当时看的我头皮一阵一阵的发麻。那一刻，我终于意识到在夜之城里，哪些才是V真正的朋友，哪些才是V真正的亲人，不管结局怎么样，曾经一起干过事情，一起反抗过命运的捉弄，就是最有意义的。
{% asset_img 5.jpg cyberpunk2077城市风景 %}
结局无疑是个悲剧，不管选择那条线，V都是要死的：最快的马上就死（V的意识消失，强尼的意识占领身体，或者是开枪自杀，提早结束这一切。）；最晚的3个月后死（从空间站回来后）。不过你可以决定让V半途而废，还是在最后的时间里，活成夜之城的传奇。这就是夜之城，这就是night city，绝望总是出现在希望之后，权力和金钱是永恒的主题。
{% asset_img 6.jpg cyberpunk2077城市风景 %}
一件很讽刺的事情是，游戏中主角V和强尼一直追求的目的（把强尼的意识单独分离出来，解决relic芯片的问题）到最后的最后，竟然是无法完成的，因为强尼和V已经“合体”太久了，导致强尼的意识深入机体的各个神经元，反而把V的意识当作了侵略方，机体不断的排斥V，反客为主了。2077是这样，现实也何尝不是这样呢？我们常常说，生活不知有眼前的苟且，还有无尽的诗和远方；可是残酷的事实是，远方除了远可能什么都没有，诗歌除了动听什么也不是。那既然如此，在这样高科技低生活的环境里，人继续前进的动力是什么呢？**赛博以至，朋克已死！！！**
{% asset_img 7.jpg cyberpunk2077城市风景 %}
2077美术设计最成功的地方在于，让人（至少有我）第一时间就感觉，欸对，世界的未来就应该是这个样子的。很符合现在时代发展的趋势。未来总有一个历史阶段会像游戏里的那个样子：科技迅速发展，各种各样的高新技术投入使用......然而这些是只有富人才用得起的东西，底层的人却仿佛还在上个时代......这就是所谓的“高技术、低生活”。高技术力的大量涌入，和巨大的社会地位、贫富差距之间的碰撞，便是赛博朋克的核心内涵之一。
{% asset_img 8.jpg cyberpunk2077城市风景 %}
很多时候我都在想，自己真要是活在2077的环境里，在那个满眼霓虹灯闪烁、灯红酒绿、科技与金钱、权力的氛围下，还能不能保持一个正常的心智，而不是变成一个赛博精神病。是的，在那样的环境里，做一个正常人太困难了，有太多的诱惑、威胁，各种各样的明争暗斗。其实不只是2077这样，在任何一个时候，做一个温柔而正确的人都非常困难，因为这个世界本身就不温柔，更不正确。
{% asset_img 9.jpg cyberpunk2077城市风景 %}
游戏玩到后面，所有大大小小的任务都做完了，等级和武器什么的也刷到了最高，百般无聊就到处闲逛看风景，开着n卡滤镜，一边欣赏夜之城绝美的风景，一边听着电脑散热扇嗡嗡作响，燃烧着显卡。这时我又开始感慨了，V成为了夜之城的传奇......然后呢？然后呢？就这样结束了？因为不管怎么想，我都实在找不到让V继续活下去的意义了（反倒是如果让强尼留下的话，最后的结局是强尼坐着车离开了夜之城，开启了新生活），反正剩下的时间也就3个月了。这也难怪，在夜之城，没有活着的传奇，想要成为传奇，只能以生命作为代价，任何人都不能例外。
游戏里很多内容都有反映现实的影子，特别是街边小人物的对话，很有意思，每次路过都得停下来听。
写到这里，不难发现2077这个游戏已经对我的思想多多少少造成了改变，之前我也不停的提醒自己，这只是个游戏而已，没有那么多深刻的问题值得思考。可是到现在我意识到，这真的只是一个游戏吗？
