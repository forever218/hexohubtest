# 正文
&nbsp; &nbsp; &nbsp; 近两个月以来，我的生活是这样的：早上11点醒，玩会电脑，然后点个外卖，或者干脆不吃，睡午觉到3点。下午去学校外面跑5公里步，回宿舍吃晚餐，晚上看书，和舍友打游戏，凌晨1点左右睡觉   
   
&nbsp; &nbsp; &nbsp;     
&nbsp; &nbsp; &nbsp; 这是什么作息？规律着来好像又不是那么健康   
   
&nbsp; &nbsp; &nbsp;     
&nbsp; &nbsp; &nbsp; 翻我高时期的某篇日记，里面写着“操！我現在的状况真是他妈糟透了！”好吧，现在看起来也不是那么糟。所以我知道，在日记里写类似的文字一点作用也没有，只会让当时的自己看起来很可笑——就这点程度吗？但是我还是想写：操！我現在的生活简直是他妈糟透了！   
   
&nbsp; &nbsp; &nbsp;     
&nbsp; &nbsp; &nbsp; 难以形容我現在的状态，真要说的话，就是找不到事干。一方面找工作的事情放下了，另一方面学校也没什么任务，距离毕设的开题报告还有一月，我不想现在就开始   
   
&nbsp; &nbsp; &nbsp;          
&nbsp; &nbsp; &nbsp; 那，我想干什么?我应该干什么？我必须干什么？我很迷茫   
   
&nbsp; &nbsp; &nbsp;     
&nbsp; &nbsp; &nbsp;“能思考能创造，这是我们区别于死人的最大之处”，这是我之前一篇小说里写的。是的，只有在做那些事情的时候，我才觉得自己是真实的，是活的。我的工作？职业？大学生？   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 每当想起这些，我便觉得自己活的虚无缥缈，不道道自己在拥挤的教室里发呆、在考试卷上乱写的意义是什么   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 我一直信奉一个道理——“为自己而活”。但是越实践越发现，这是很难的、甚至是不可能的事情。比方说，在干每件事之前都先问自己，这件事真的是自己想干的吗？还是希望别人知道自己干了？   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 抱歉，过去的我，我真的做不到。我没法完全不顾他人的想法行事，仔细想想，上述的问题并没有多少少情况是前者。   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 人是社会动物，我很难罢脱这种需求：别人的赞美与认同。但我会尽可能这么做——跨越跃那些拥抱过我的泥沼   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 我写作，我写代码，我画油画，我摄影……我想以此来留下我活过、思考过的证明。但是最近，我似乎对它们都失去了兴越。真该死！我到底是怎么了？   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 从这个角度看，我已经死了两个月了   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 你说我“堕落”，我觉得一点问题也没有，事实就是如此。好比一件白衣服，如果沾上了一点污渍，我便会想：反正都脏了，那为什么不再脏一点呢？   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 入们总是等一件衣服脏到了一定程度再去洗      
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 球队的人喊我去打球，我说“我累”；舍友叫我打游戏，我说“不打”；同学拉我去看电影，我说“你们去就行”。我想成为一个有趣的人，但同时我也问自己：这不就是为别人而活了吗？   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 归根结底，也许我还是一个无聊的人，只是这段时间摘下了面具，仅此而已   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 每个人都有虚伪的一面，如果你能看见那个阴暗的我，无论你的想法是什么，我都不胜感激   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; “书藉是最好的解药”，至少现在并非如比。我已经对这剂解药产生了抗药性。我发现，我从一本书里汲取到的负面思想，远比正面思想要多得多   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 哈哈！为什么你总是看不到好的那一面呢?   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 有一次，我花了一天时间坐在教室里看《百年孤独》，不吃饭，不喝水，不休息，从早上看到晚上。关掉电子书我便头晕目眩，差点在教室里呕出来   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 操！我在干什么！闭上眼，我仿佛是布恩迪亚家族的一员，魔幻、虚无   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 马孔多又下雨了   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 这样的生活让我怀疑自己存在的价值   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 好吧！既然不知道干什么，那就看看身边的人在干什么，我这样想着，打开朋友圈。旅游、美食、谈恋爱，各种打卡自拍，几乎没有其他的   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 别人的生活比我的精彩多了   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 真的是这样吗？   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 我看到的，只是别人想让我看到的   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 我从不刷短视频，身边同龄人谈论的很多话题我都听不懂，但那不重要，我也不在乎，只是一味地看各种乱七八糟书，尽管这会加重我现在的症状   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 前几天我在一位日本朋友的博客上看到这样一句话   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 本を読まないということは、そのひとが孤独でないという証拠である   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 不读书证明你不孤独   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 我才忽然意识到，原来自己的病症也许是孤独。病因是什么？就像发烧的病因是细菌感染之类的，我肯定也自己的病因。可惜我不知道，医者不能自医   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 晚上我躺在床上，脑子里想着一堆事情，不切实际的、想入非非的。我在想什么？   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 高海拔地区的动物、植物平均都要比低海拔地区更高大，是不是和气压有关系？高海拔气压低，生物向上生长的压力更小   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 看到过一个很经典的引力模型：一块展开的布，放上一个大铁球，再放一些小铁球，小铁球便会旋转着，最终落向大铁球。那为什么月球在远离地球？   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; ……诸如此类   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 很可笑是不是？哈哈！你连最基本常识都没有！我当时真的是这么想的，大概脑子已经闲出毛病了   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 嗯？Github上有新issues了？有新的PR了？有BUG要修了？有实验报告要交了？我不想管，我什么都不想做，我累了   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 上周日晚上，我突然想起这个星期还没自慰，再不撸要遗精了。妈的，我才不想早上去洗内裤！   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 好，那就睡前撸一发吧   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 然后我爬上床，在头刚接触到枕头的时候便昏迷了过去   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 操，你怎么睡着了！   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 哈哈哈！连看片都没有性欲，自慰都做不到了吗？   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 我猜我真是最差劲的   
   
&nbsp; &nbsp; &nbsp;   
&nbsp; &nbsp; &nbsp; 2025.12.25  于中北大学
