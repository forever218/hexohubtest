# 找工作
&nbsp; &nbsp; &nbsp; 我总是认为，人工作了某种程度上就是死了，班味很大一部分就是死气沉沉的，失去了曾经对生活和兴趣爱好的热爱（好几个学长学姐工作中之后眼睛都无神了）。工作和生活之间的关系是什么？他们是否互相对立？工作是否只是生活的一部分？这是一个非常主观的问题，同时也决定着以后的生活方式，我会寻找自己的答案。  
   
&nbsp; &nbsp; &nbsp; 最近一直折腾找工作，都说“金九”，九月是拿offer的最佳时期，但是秋招到现在，我身边绝大部分同学都没找下，我也找不到一点。总而言之，我的九月乃至到现在，都是在`投简历`-`面试`-`笔试`-`投简历`这样的循环中度过的。一开始干什么都还是满怀希望的，直到后来逐渐麻木，相比较那些投完简历、面试之后便再没下文的，甚至感觉收到拒绝的邮件对于我来说都是一种恩赐（不知道心理是不是已经出问题了）。被拒的太多了，这里就贴一个最近的：  

{% asset_img d85180d850a0e103a2c1749f93dcea2e.jpg "图片描述" %}

&nbsp; &nbsp; &nbsp; 之前去了好几次大型的双选会，本校的、外校的都有，感觉能投的岗位并不多，北方企业一般招的都是能源、化工、电子、材料这样的专业。其实单看公司名就知道跟我没关系了：  

{% asset_img 31db70fe0be8b721e2bc259b159bd070.jpg "图片描述" %}

&nbsp; &nbsp; &nbsp; 感觉还是要往南方靠，对口的岗位才多一点。
&nbsp; &nbsp; &nbsp; 而且去双选会、宣讲会的时候，身边很大一部分都是研究生，我就在想，研究生都还没找到工作，自己身为本科生又凭什么找得到呢，想想就苦笑，211、985的研究生；双非的研究生；211、985的本科生；双一流的研究生、本科生，哪个不是在我前面，“金九”估计只是给他们准备的，普通一本的本科生只能到最后捡吃剩的（除非是特别厉害的那种）。    
   
&nbsp; &nbsp; &nbsp; 面试的时候，HR总会问我：为什么不回广州去呢？真是一个很好笑的问题，要是我能在南方找到工作，我还投你的应聘干嘛。南方来北方校招（线下）的企业本身少之又少，只能走网申，并且网申周期又长，我只能尽量在北方地区也投，填满等待的空窗期，同时也能增大就业的几率。   
    
&nbsp; &nbsp; &nbsp; 都说找工作难，到自己真正亲身经历了，才知道**难**的具体含义。其实吧，从某种角度来说，确实不难：  

{% asset_img fc07c49e8259f9a3bf550f60a5746eae.jpg "图片描述" %}

要是放得下身段，随便就能找一个，但谁又会甘心呢？     
    
&nbsp; &nbsp; &nbsp; 前几天去一个宣讲会，我站在最后面，看着下面密密麻麻的求职者，都举着各自的简历，突然想到和纪录片上看到的蚂蚁一模一样（叫切叶蚁还是什么，不记得了）：

&nbsp; &nbsp; &nbsp; 两者之间还有一个共同点：一踩就死。~~骗你的，我们本科生没这么坚强，不踩也会自己死~~   
     
&nbsp; &nbsp; &nbsp; 一个学姐最近安慰我，说不着急，慢慢来，找不到工作不是我们的问题，是时代的问题。想想也是，工作总会找见的，水到桥头自然直，就像雨不会一直下，不过雨后不一定有彩虹罢了。关于“时代的问题”，我也想了好一会——**时间总会治愈一切，那时间也病了呢？**

# 饥饿
&nbsp; &nbsp; &nbsp; 很久没有体会饥饿的感觉，昨天下午狠狠体验了一把。昨天一整天就早上吃了点东西（中午在实验室待着不想出门），然后下午出学校跑步，跑了4公里之后肚子绞痛的不行，那种极度饥饿带来的痛感，非常明显，很好辨别。然后我就慢慢走回学校，一路上时不时一阵反胃（但是吐不出东西来），头晕目眩的，几公里的路感觉相当漫长。并且这几天降温明显，5点基本上就6度左右，还是阴天没太阳，估计最后一段路是腺上肾素发力，要不真走不到饭堂就晕倒了。到了饭堂整了一份大份云吞，猛猛炫完（其实主要是先灌汤，流体能迅速撑满胃部，高中的经验），回宿舍就没啥事了。   
    
&nbsp; &nbsp; &nbsp; 现在回忆起来，我第一个想到的是可惜没带运动相机（我出门跑步都不带手机，只带一个运动手环），不然把当时难堪的状态记录下来。

# 地震
&nbsp; &nbsp; &nbsp; 昨天中午，我在床上躺着准备睡觉，突然床剧烈晃了一下，我下意识就喊下铺别抖腿，睡觉呢。但翻身一看下铺不在，嗯？什么鬼东西，当时感觉自己是不是做梦，半梦半醒的出幻觉，就没管了。睡醒才知道是山西地震了。

# 服务器
&nbsp; &nbsp; &nbsp; 从没在[腾讯云](https://cloud.tencent.com/)上买过服务器，上一个服务器还是在阿里云买的。前几天在腾讯云上面改域名解析的时候，看见一个新手特惠，4核4g的服务器一年只要79（40G硬盘，3M带宽，300G月流量），直接一个没忍住下单了，冲动消费说是。   

{% asset_img b819727d728a6271a08c383caa2d77b5.png "图片描述" %}
   
&nbsp; &nbsp; &nbsp; 买了之后一时间不知道要拿来搭什么，但有个服务器就总想把他的算力榨干（估计过一段时间就吃灰了哈哈哈），博客系统的话也有hexo和免费的托管在支持，自己也懒得再去折腾。  
   
&nbsp; &nbsp; &nbsp; 目前在上面运行着两个服务，一个是私有网盘系统，一个是内外穿透（映射一些本地项目服务的端口）。总体来说体验还是非常不错的，私有小网盘省去了传文件要登微信的烦恼，和同伴共享一些视频、图片也非常方便，5g以内的文件都能在网页端快速传输。唯一限制的就是3M的带宽，如果网盘的存储策略直接走服务器的话，可能速度有点慢，换成又拍云之类的存储策略就好很多，网速能跑满。
