# 教程

## 安装所需环境

项目所需环境为`Nodejs 6.0以上版本`,`Git`;
- 安装`Nodejs`:https://nodejs.org/en
- 安装`Git`:https://git.p2hp.com/

一路傻瓜式的安装，一直点确定就可以了。

## 安装cnpm

`cnpm`是国内`npm`的镜像包。在这个项目中，用`npm`会出现奇奇怪怪的问题，所以这里我们选择用`cnpm`：

在桌面上右键，选择`git bash here`，执行命令：

``` bash
npm install cnpm -g --registry=https://registry.npm.taobao.org
```
即可完成安装。如在过程中遇到问题，参见：

- https://www.jianshu.com/p/fbd87ad45631
- https://www.oschina.net/p/cnpm?hmsr=aladdin1e1
- https://npmmirror.com/

## 安装项目

在合适的地方（这里以桌面为例）。右键，选择`git bash here`,执行命令：

``` bash
git clone https://github.com/Tomotoes/HomePage.git
```

这时桌面上会出现一个名为HomePage的文件夹。
### 修改错误并安装
`项目有些地方有出错，接下来需要我们手动修改：`
进入HomePage文件夹,找到`package.json`文件，将其中的`node-sass`和`gulp-sass`所在行全部删掉。
回到HomePage文件夹，右键选择`git bash here`，分别执行命令：
```bash
cnpm install node-sass --save-dev
```
```bash
cnpm install gulp-sass --save-dev
```
再找到`gulpfile.js`文件，将第12行改成这个：
```javascript
const sass = require('gulp-sass')(require('sass'))
```
然后回到HomePage文件夹，右键选择`git bash here`，执行命令：
```bash
cnpm install sass --save-dev
```
```bash
cnpm install
```
至此，项目安装完成。

## 启动、预览和部署
在HomePage文件夹下执行以下命令以启动预览：
```bash
cnpm run dev
```
在HomePage文件夹下执行以下命令以启动构建：
```bash
cnpm run build
```
构建完之后`dist`文件夹里的静态文件，即可上传至服务器完成部署。

## 配置

修改HomePage文件夹下的`config.json`文件来完成个人信息定制。

更多信息请参见项目文档：https://github.com/Tomotoes/HomePage
