# 见闻


&nbsp; &nbsp; &nbsp; 新闻说，xxx工厂旧址改造成了一个xxx主题公园，老爸看了说：不知道油制汽厂能不能像这样改造？嗷，那里已经拆掉了，新建了一大片住宅区，解散换岗后我也好久没去过了。建小区之前，人们把表层的土壤全部铲起来，加热之后晾在另一个地方，又把很深很深的土翻到上面来。听我爸说，当时翻上来的土都黑的发亮，而且散发着一股煤油的气味。
<hr/>

&nbsp; &nbsp; &nbsp; 在增城小楼镇，有一棵树长在了烟囱上，当地人叫它“百年飞榕”。
<hr/>

&nbsp; &nbsp; &nbsp; 听我学姐说，现在“学术”圈子乱成一团，大家为了保研的名额争破了头，互相举报来举报去的，都在爆料对方的“黑历史”，自己保不上别人也不能上，所有人都别想好过。
<hr/>

&nbsp; &nbsp; &nbsp; 爷爷说，他以前在的连队总共有40个人，已经走了22个，今年战友会能来的有7个，另外有3个脑溢血，2个在家里瘫痪。他说，参军之前，他年轻没事做，放学的时候就去爬高崎寨（梅州的一座山），以前上面有个解放军建的通讯铁塔，他就爬到上面去，能望见整个梅州城。
<hr/>

&nbsp; &nbsp; &nbsp; 春晚节目《春意红包》，有人说真不赖，有人说这跟韩国女团有什么区别。
<hr/>

&nbsp; &nbsp; &nbsp; 奶奶说，很久很久以前，她在田里插秧，爷爷就从旁边路过，爷爷对她那是一见钟情，后面死缠烂打，奶奶就嫁给了他。她说，这辈子最大的福气就是跟了爷爷，爷爷年轻时当了兵，回来之后在镇上卖猪肉，又把奶奶的户口从农村转到了城市，这在以前都是很了不起的。
奶奶说，以前哪知道谈恋爱，都是结了婚再谈恋爱的，她生我爸的时候才22岁，仿佛把90年的一生在30岁就过完了，时代啊，不同咯！
<hr/>

&nbsp; &nbsp; &nbsp; 妈妈说，她对于死亡的恐惧来自于未知，死后有什么感觉？死后会去哪里？以前她对死亡没有概念，直到外公离去，她才真正感受到了生命的震撼。
<hr/>

&nbsp; &nbsp; &nbsp; 爷爷说，他的眼睛是在对越自卫反击战中受伤的。当时他在前线挖战壕，头顶上炮火纷飞，突然他看见什么东西掉进了前面的树林里，然后就失去了知觉。醒来后已经是在后方的医院里了，左眼裹着纱布，幸运的是，爷爷的左眼并没有失明。
