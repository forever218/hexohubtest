关于梦想，可谈的东西太多了。我只能从仅限的几个角度，开始我的想入非非......
{% endnote %}
  很小很小的时候，老师也好，父母也罢，都会经常问我们：你的梦想是什么？小学的同学在老师问的时候都说：将来当消防员，当老师，当医生......问到我，我就说：我的梦想是拥有一个梦想。这不是开玩笑话，当时我就是这么讲的，估计是把老师吓了一跳。童言无忌，我没有梦想，就是没有梦想，绝对不会随便附和说自己想当一名警察。不仅是当时，现在我也觉得当一个什么什么不能算是一个人的梦想，大多数时候人们对于“你的梦想是什么”这个问题都有点答非所问。当一个什么（职业），只能算是一个目标，而绝非梦想。有关两者的区别，我是这样理解的：目标是短期内有很大概率可以实现的，是个人的；梦想是在长期内都很难实现的，不一定是个人的。比如说：我的梦想是人类统治太阳系，这里的主体就是“人类”，和“我”没什么关系。
{% note info simple %}
现在再来看这里，似乎有点牵强。梦想也不一定是我说的那样不可到达，这取决于我们怎么去实现。
{% endnote %} 
  必须要说明的是梦想是可大可小的，但没有高低贵贱之分。有的人梦想看起来宏伟，但说到底却也是满纸空谈；有的人梦想看起来好不起眼，却能做到真正的伟大。当然，伟大也好，幻想也罢，到底如何来评价一个人的梦想毫无依据可言（一些阴间梦想除外），这与时代和社会环境有着很大关系。
  在谈我自己的梦想之前，我发现可以将过去的经历分为两种心境：有梦想的，无梦想的。这两种情况交错在一起，非常混乱。有一段时间，我的梦想是成为林丹那样的羽毛球运动员；后来变成当一名科学家，发现万物的规律；再到后来，是想让人类走出太阳系......总之，各种各样的梦想我都有过，离谱的、不切实际的、脚踏实地的、想入非非的、正常的、大众的、前所未有的、道德败坏的、胸怀天下的、自私自利的......只是到最后，依旧没有一个明确的想法。按理来说，“做梦”的频率会随着年龄的增长而减小，我自己倒是个例外：从幼儿园到大学，无时无刻都在更新着各种各样奇怪的梦想。有句名言说：“梦想家短命，实干家长寿”，让我恐慌了好一阵子，因为我知道自己做梦多于实干，生怕哪一天会挂掉，比别人短命。当然这是那时候的想法，现在长大了，不会那么无厘头了。
{% asset_img 1.jpg This is an example image %}
  据我观察，身边的人大多数也有这样的情况，不清楚自己的梦想到底是什么。绝大多数想的是将来有一份体面的工作，稳定而高的收入，幸福的生活。这个梦想很棒，一点问题都没有，我的想法也是这样的。可我总是觉得，这样的目标还是离所谓“梦想”有一定距离。真正的梦想也许是像艺术那样，源于生活而又高于生活，既不能太切合生活，又不能太脱离脱离实际。换句话说，现实与幻想的交点，谓之梦想。
  袁隆平的“稻下乘凉梦”我认为就是一个非常出色的梦想，很大程度上也是他如此成功的原因之一。这个“稻下乘凉梦”很完美的找到了现实和幻想的平衡点，尽管可能很抽象，结果很含糊，但它却真真正正达到了心目中梦想该有的高度。
  伟人说：“我有一个梦想......”，我们就觉得很牛逼；而身边的普通人说：“我有一个梦想......”，我们却觉得很傻逼，这是普遍的现象。归根结底还是在于我们怎么去看待梦想。伟人的梦想是梦想，平凡人的梦想也是梦想，都体现着个人的人生追求和理想，都值得别人去尊重和理解。


  直到现在，已经是一个大学生了，还是不知道自己的梦想究竟是什么，这点让我非常苦恼，经常失去继续努力下去的动力。当然，我还是有着各种短期的目标，比如考试成绩要到多少多少，要加入几个社团。这充分说明，目标只能给人以短期的动力，时间长了，人就会懈怠。能至始至终激励人走下去的，还得是梦想。
  想来想去，对自己失望至极，简直他妈废物一个。没有梦想，浑浑噩噩。希望这样的状况在未来有所好转，因为现在真的太难受了，不知道下一步要干什么，非常迷茫，有事没事不停的问自己：然后呢，然后呢，然后呢......
  诚然，一个人可以拥有好几个梦想，虽说不是多多益善，但也未尝不可。而我的问题在于，自己根本分不清哪些梦想是正经的，哪些是想入非非的；哪些是自己真正向往的，哪些只是一时的热血。没有梦想的人，谈梦想都费劲。
  这几天看的《唐吉可德》（商务印书馆·刘京胜-译）里有这么一句话：一个有梦想（原文为“理想”，我认为理解为梦想也是同义）的人，它带领我们忍受折磨，穿越苦难。没有梦想的人则是空洞、单调而又不丰满的，但梦想不切合实际抑或空有想法而不付诸行动，终不能收获喜悦和果实。
