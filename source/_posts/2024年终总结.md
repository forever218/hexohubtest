# 前言
{% asset_img 0.jpg  %}
&nbsp; &nbsp; &nbsp; 2024年已经过去，遗憾也好，欣喜也罢，于我而言，这都是意义非凡的一年。2024，我做了很多？收获了很多？好像都没有，只不过是一个普普通通的大学生，从大二升到了大三，离步入社会又进了一步而已。
&nbsp; &nbsp; &nbsp; 前几天我跟朋友聊起这个，他说看看自己记录的东西，好像2024也没那么不堪，起码在玩。想想很有道理，2024年没有什么大事发生，自己也是在到处瞎玩。只有把自己做过的事情记录下来，回过头看的时候，才会有莫大的成就感——哦，原来自己也没有那么碌碌无为。

## 意义
&nbsp; &nbsp; &nbsp; 2024年，花了很多时间去思考“**意义**”这个词，从过去到未来，从我自己到整个宇宙......这么说有些滑稽（期末考挂科前的幻想罢了）。这些本都是虚无缥缈的念头，自然也不会想出什么实际的结果。
{% asset_img 4.jpg  %}
&nbsp; &nbsp; &nbsp; 具体来说，就是经常会问自己，我为什么要做这件事？这件事对我有什么意义？这件事对......好比一个无底洞，把我的思维狠狠的困在里面。
&nbsp; &nbsp; &nbsp; 或许就像《盗墓笔记》里面说的，意义这个词，本身就没有意义。过去的经历，成就了现在的我，现在的我，又会成就未来的我。这个过程，就是意义。

## 旅途
&nbsp; &nbsp; &nbsp; 所谓旅途，无非就是去不同的地方，见不同的人，做不同的事，明白不同的道理。2024，最有意义的一次旅行，应该是去香港。
{% asset_img 28.jpg  %}
&nbsp; &nbsp; &nbsp; 旅途的意义又是什么呢（接回上一个章节）？我想了很久，觉得自己生活之旅的最大意义便是“**不确定性**”，将来的生活，未来的时代，未知的科技，都充满了不确定性。正是这种未知数，给了我继续旅行下去的动力。
{% asset_img 6.jpg  %}
&nbsp; &nbsp; &nbsp; 这也是为什么我不想当老师、公务员这样类似职业的原因，一眼望到头的生活，这辈子真的就那样了。当然，这也不是说这些职业不好，只是我个人不喜欢而已。人生是旷野，我更愿意迎接无数的未知。
&nbsp; &nbsp; &nbsp; 既然是旅途，路上的同伴自然是来来去去的。年中的时候，大四的（比我大两届）学长学姐毕业了，他们都是非常优秀的人，我也很羡慕他们。当时只是简单的说两句拜拜，我也总不能把我的不舍撕心裂肺的表达出来，但我知道，这一再见，可能真的就是再也不见了。世界太大，人生轨迹的焦点少之又少，能在大学交汇，已经是莫大的幸运，我不能再奢求什么。
{% asset_img 2.jpg  %}

## 成长
&nbsp; &nbsp; &nbsp; 我现在20岁多一点，已经是奔三的人了。周围的一切，家长、老师、社会，都在无时无刻的告诉我，我应该更加成熟，按照一个大人的标准来行事。唯独我自己一点也不想长大，只想无拘无束，无忧无虑的用自己的方式生活在这个世界上。
&nbsp; &nbsp; &nbsp; 我也明白，很多事情不是我能决定的。为此，我能做到的是，尽量在一些重要的事情上表现的成熟一点，其他时候还是那个不成熟的我。妈，还有以前的羽毛球教练，学校的老师，都说我怎么老是嘻嘻哈哈，一点都不正经，分不清什么是正经，什么时候该严肃。
{% asset_img 5.jpg  %}
&nbsp; &nbsp; &nbsp; 对于“成长”和“正经”，我有和他们不一样的理解，成长不是变得严肃，成熟也不是变得正经，而是变得更加自信，更加独立，更加有自己的想法。有责任心，有担当，有勇气，有智慧，这才是真正的成熟，而不是一天到晚板着个脸做一些没有意义的东西。
&nbsp; &nbsp; &nbsp; 至于为什么老是看起来玩世不恭的样子，有两点解释。第一，是我真的已经很长时间没有感觉到焦虑和紧张了（甚至高考也没有），可能是因为自己把很多东西想的太通透，还没开始就已经看到了结局。第二，也是我的人生准则之一，永远不要把自己的负面情绪影响到别人。所以过去的十几年里，我没有大动干戈的发火什么的，凡事发生皆有利于我，不好的想法和情绪应该自己消化，把积极的一面留给其他人。
&nbsp; &nbsp; &nbsp; 另一方面，觉得自己成长的同时，也在变得麻木。这绝不能说是一件好事，但确实在悄悄地发生着。很多时候，发生了一些事情，都会觉得无所谓，不会有太大的情绪波动。正在努力的去改变这种状态。

## 记录
&nbsp; &nbsp; &nbsp; **记录**是我2024年的一个关键词，记录自己的生活，记录自己的思考，记录自己的创造。2025我的目标也是一样，做一个忠实的生活记录者。2024，写了50篇日记（平均来看都快成周记了），随身携带的运动相机(DJI action3)记录了460G内容。

&nbsp; &nbsp; &nbsp; 主要看的三类书，一是**哲学**，二是**科幻**，再就是**小说**。哲学（philosophia）真的很有意思，由“philo”和“sophia”组成，直译就是“爱的智慧”，总是能站在不同角度看待平凡的事物。或许是随着年龄的增长，经常开始思考死亡和生命，所以看哲学方面的书主要还是围绕这两个主题。
&nbsp; &nbsp; &nbsp; 后来突然发现，无论是生命还是死亡，似乎都离不开**性**，于是又去看关于性的书，其中不乏年代久远的作品。说到性，必然又离不开**爱**，又跑去找书看，想想明白这两者之间的联系，究竟什么是**性**，什么是**爱**呢？到现在我还是一头雾水。
&nbsp; &nbsp; &nbsp; 总而言之，2024年看了一堆乱七八糟的书，除了上表列出的，还有很多像《读者》和《中国国家地理》这种杂志也看了十几期（每月订阅）。真要说有什么收获，大概就是减少一些无聊的时间，顺便提升了一下自己想入非非的能力吧。之前屯下来的一些《中国国家地理》和《读者》：

## 总结
&nbsp; &nbsp; &nbsp; 如上文所说，2024年思考了一大堆有的没的、没有什么实际价值的东西，以至于到年末（直到现在）甚至都怀疑自己精神状态是否正常，还是说这是精神内耗的另一种表现形式罢了。
{% asset_img 30.jpg  %}
&nbsp; &nbsp; &nbsp; Anyway，2024过的还不赖吧，嘻嘻哈哈，玩玩闹闹。**思考**，**折腾**，**创造**，**学习**，**记录**，***在迷茫中前行，在不确定中成长，这就是我的2024***。
