&nbsp; &nbsp; &nbsp; 花了点时间看完了弗洛伊德的《性爱与文明》，感觉似乎与书名不太相符，书中花了大量篇幅探讨了“性”和“爱”之间的关系，而非“性”“爱”“文明”三者之间的联系。
&nbsp; &nbsp; &nbsp; 首先有个很有意思的问题：为什么人类，或者说现在的大多数生物，会分为两性呢？AI回答大致可以概括如下：能使后代获得新的基因组合，适应环境的能力更强，清除有害突变，是自然的选择。
&nbsp; &nbsp; &nbsp; 或许“2”这个数字在配偶效率和后代质量之间达到了某种平衡，如果是3性繁殖，虽然后代质量会提高，但相应的减数分裂成本也会提高。我认为在生物进化的历程中肯定有生物进行了3性繁殖的尝试，不过显而易见的，被自然所淘汰了。
&nbsp; &nbsp; &nbsp; 人类之间的两性关系远比其他生物要复杂的多，根本原因是人类是一种高度智慧的生物。性，早已被赋予了不同于繁殖后代的意义。现在大多数做爱的目的都不是为了生育，而真正为了生育目的而做爱的占比是相当小的。
&nbsp; &nbsp; &nbsp; 另一个有趣的现象是，女生经常通过漂亮的打扮来吸引男生，吸引的程度通常通过“性感”来描述，然而就我的观察来看，男生似乎对自己如何靠外观来吸引女生很没有把握。举例而言，一个身材不错的女生穿着短裙丝袜高跟鞋，那么就可以用“性感”来形容，对男生的吸引力是相当大的，甚至对女生的吸引力也不小；而男生就好像难以用“性感”来形容。究竟什么样的穿着才能算上性感呢？换句话说，女生很容易就是能让男生产生性欲，反之男生却很难让女生产生性冲动。关于外表方面有一个很生动的例子是，在几乎所有的日本动作片中，男性都是光着身子的，而女性都有一些象征着“性感”（或者直接说“性”）的服饰，比如前面提到的高跟鞋之类的。
&nbsp; &nbsp; &nbsp; 上述的现象能否说明一个问题？男生的性感或许不是主要体现在外表穿着上面。作为一个男性而言，性感一词有着更深层次的含义，当然了，这是一个看脸的时代，有一个优秀的建模，肯定是有加分的，只不过可能穿着对男性来说不像女性那么能展现出性的吸引力，要不然男优怎么只会裸体呢。
&nbsp; &nbsp; &nbsp; 我必须要说明的是，上面我对于性感的论述既不客观，也不基于全部事实，仅是我的片面经验之谈。我觉得这种两性的差异很有意思，即在引起性冲动的外表方面的不对等，这是一个很复杂的问题，我也不可能研究得清楚，但我隐约感觉到，这可能跟自人类步入文明社会以来男女之间的分工有关系。
&nbsp; &nbsp; &nbsp; 书中一个我很欣赏的观点是“性”（或者说原欲）是一个人与生俱来的本能和属性，是一个人最基本的冲动和欲望，而不是在某个人生阶段才有的所谓的性的萌芽，例如青春期。幼儿时期的种种表现从本质上来说都跟性有相似的地方，例如喝母乳和吮吸手指，前者有实际的生理意义，而后者则是通过吮吸黏膜来获得快感——一种最简单的性的满足方式。
&nbsp; &nbsp; &nbsp; 事实证明，儿童对于性的认知超过其父母的想象，就像一个几岁的儿童很容易就能看出其母亲是否怀孕那样。只不过受限于表达和交流能力尚不完备，无法和外界解释而已。长期以来，性教育是普遍缺失的，受到的风俗习惯的影响，在教育者看来，“性”都是不好的东西，是罪恶的源泉，所以在教育过程中总有意的避开这个话题，这就导致了幼儿对于性的探索完全是无目的、的自主的。所以，我认为童年时期也是塑造一个人性癖最关键的节点，这个时间段内所有对性的无目的的探索都会成为其成年后“性”的基石
&nbsp; &nbsp; &nbsp; “一般的说，我们的文明乃是根基于对本能的压制上的......”，人类之所以能称之为文明，很大程度上也体现在对性的压制上，我们制定某种规则不让性泛滥成灾。但一个有趣的问题是，性的本质是为繁殖后代服务的，而文明的首要任务是生存和发展（或者说扩张），从这个角度看，对性的压制似乎是文明发展的阻碍。
&nbsp; &nbsp; &nbsp; 对此，我能想到最合理的解释是与自然环境有关。从很早以前的某些时候，人们发现身边的资源无法满足如此多后代的需要，便开始了对性的压制，这种压制完全是出于实际需要的，与现代的性压制有本质区别，这种压制一直持续到某个历史节点，最后演变成了规则与风俗习惯，某种程度上来说也导致了现在的性压抑社会。
&nbsp; &nbsp; &nbsp; 现在有研究表明，幼儿时期与父母的爱可能多少带点新的异味，如婴儿吮吸母乳可以看成一种性行为。这样一说，父女母子之间圣洁的爱好像一下就污渍了起来，我们先抛开这个问题不谈，我想说的是，上述研究似乎证明了性与爱之间的某种联系，即他们不是独立的，或者可以这样说，爱是精神层面的性，而性是对爱的肉体实践。
&nbsp; &nbsp; &nbsp; 就我而言，我觉得这个观点并不能很好地解释现代的性爱关系，我的观点是**性爱分离**，性与爱之间或许并没有太多联系，可以说是独立存在的。爱对象是否是性对象？性对象是否是爱对象？这是一个很主观且复杂的问题，每个人的答案都不一样，不过一个普遍的现象是，爱很容易就能产生性（比如说情侣），反之性却不一定会产生爱（比如说一夜情）。随着文明的进步，我认为性爱分离的趋势会愈加明显。原因很简单，从进化的角度看，性行为与生殖本能相关，并不需要“爱”这样的高级情感来绑定，就像动物界的性行为基本上与爱无关那样。爱是人类作为高级智慧生物的一种情感，只不过在文明进程的某个时期与性联系在了一起而已。
&nbsp; &nbsp; &nbsp; 所以从生物进化的角度来说，可以没有爱，却不能没有性。但如果这样片面的做结论的话，会淡化了“爱”在“性”中的重要性，人类社会中性与爱之所以有如此多的联系，肯定是有其原因的。性是人类得以存在和繁衍后代的基础，这是一个不争的事实；人类文明离不开爱，这也是一个不争的事实；上述两个事实说明，即使性与爱是独立存在的，对人类而言都有着不可或缺的重要地位。没有了性，人类就无法繁衍后代；没有了爱，人类便与野兽无异。至此，我的结论是：`性是人类生生不息的源泉，而爱则是人类文明存在的根基`
