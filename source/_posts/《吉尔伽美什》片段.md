这几天看完了《吉尔伽美什》这部史诗，大为震撼。作为现存已知最早的一部英雄史诗，它在历史、文学上理应有着极高地位，但似乎对于它的研究比起其他史诗体系而言又是极度缺乏的。
诗歌中有些地方真的写的很好，在现在看来依旧有着相当的水准，很能打动人。我把一些好看的片段摘录下来.其中[]是原泥板已经损坏或是无法辨认。

{% note primary simple %}
此人见过万物，足迹遍及天边，
他通晓一切，尝尽苦辣酸甜，
他和[  ]一同[  ].
他将睿智[  ]将一切[  ].
他已然获得藏珍.看穿隐秘，
洪水未至，他先带来了讯息，
他跋涉千里，归来已力尽精疲，
他把一切艰辛都刻进碑石，
他修筑起拥有环城的乌鲁克城墙。
{% endnote %}
{% note info mordern %}
原文如此，“他”指的是吉尔伽美什。我怀疑石碑（另一说泥板）是否真的是他自己所刻。
{% endnote %}
{% note primary simple %}
我的朋友啊，谁曾超然人世升上了天？
在太阳之下永生者只有神仙，
人的寿数毕竟有限，
人们的所作所为，无不是过眼云烟！
你在此竟怕起死来，
你那英雄的威风为何消失不见！
让我走在你的前面！
{% endnote %}
{% note info mordern %}
这是吉尔伽美什劝勉恩奇都的话，体现了他自己对于生死的看法。
{% endnote %}
{% note primary simple %}
领到那个家,有进无出,
领上那步行者有去无回的路，
领进那住人的家，光亮全无。
{% endnote %}
{% note info mordern %}
对阴间的描写
{% endnote %}
{% note primary simple %}
难道我们能营造永恒的住房？
难道我们能打上永恒的图章？
难道兄弟之间会永远分离？
难道人间的仇恨永不消弭？
难道河流会泛滥不止？
难道蜻蜓会在香蒲上飞翔一世？
太阳的光辉岂能永照他的脸，
亘古以来，便无永恒的东西。
酣睡者与死人一般无二，
他们都是一副死相有何差异，
神规定下人的生和死，
却不让人预知死亡的日期
{% endnote %}
